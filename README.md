# وعي المستثمر الصغير 🛡️

موقع **وعي المستثمر الصغير** — منصة تعليمية لحماية المستثمرين الجدد من الاحتيال الرقمي.

## 🚀 رفع الموقع على GitHub Pages

1. أنشئ repository جديد على GitHub (مثلاً: `investor-awareness`)
2. ارفع محتوى هذا المجلد كاملاً
3. اذهب إلى **Settings > Pages**
4. اختر `main` branch ثم اضغط **Save**
5. موقعك سيكون على: `https://username.github.io/investor-awareness`
